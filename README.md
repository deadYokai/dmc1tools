# DMC1 Resource Repacker

## Depency:
* Python 3

## Usage:
* ### Fsd/Dat/Itm Files:

   #### Unpack:
    ```bash
    $ python3 dataRepack.py <file>
    ```
    #### Repack:
    ```bash
    $ python3 dataRepack.py -p <file>
    ```

* ### Texture Files:

    #### Unpack:
    ```bash
    $ python3 textureRepack.py <texturefile>
    ```
    #### Repack:
    ```bash
    $ python3 textureRepack.py -p <texturefile>
    ```

    #### Supported texture files:
    - .tm2
    - .ip2
    - .t32
